# ⛏️ Harven Mine Predic

Harven Mine Predic est un mini site web conçu pour fournir des prédictions sur les mines à partir de données et d'algorithmes d'analyse. Son objectif est d'offrir une estimation des probabilités, tout en restant simple et accessible.

## Fonctionnalités

- 📊 Prédictions rapides
- 🎨 Interface simple et intuitive
- 📱 Compatible avec les appareils mobiles
- ⚡ Léger et facile à utiliser

## Installation

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/votre-utilisateur/Harven-Mine-Predic.git
   ```

2. Ouvrez le dossier du projet.

3. Lancez simplement le fichier `index.html` dans votre navigateur.
